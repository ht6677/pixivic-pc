# 简介

一个简单的pixiv图片工具站，前端由vue全家桶驱动。

## 项目特点

- [ x ]  流畅的瀑布流
- [ x ]  图片详情模块
- [ x ]  画师模块
- [ x ]  用户模块
- [ x ]  评论模块
- [ x ]  搜索模块
- [ x ]  排行模块
- [ x ]  画集模块
- [ x ]  多语言i18n

## 启动项目

### clone

```
git clone https://github.com/cheer-fun/pixivic-pc.git
cd pixivic-pc
```

### yarn

```
yarn
yarn serve
yarn build
```

### or npm

```
npm install
npm run serve
npm run build
```
